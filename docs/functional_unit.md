# Functional Unit (Frozen)

Functional unit:
**1 MWh useful heat delivered to the district heating (DH) delivery boundary.**

DH delivery boundary definition (part of the functional unit):
- Supply temperature Ts (K)
- Return temperature Tr (K)

Notes (frozen meaning):
- NOT 1 MWh stored
- NOT 1 MWh input
- ONLY useful heat delivered across the DH boundary
